# foltone_dialog

## Description
Script qui permet de saisir un texte dans une fenêtre de dialogue en NUI.

## Utilisation
```lua
exports["foltone_dialog"]:FoltoneDialog("Test", "Enter message", function(result)
    print(result)
end)
```
